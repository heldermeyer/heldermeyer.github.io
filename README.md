# Professional Portfolio | Helder Meyer

Repositório contendo o código-fonte do meu portfólio pessoal e profissional. Este projeto foi desenvolvido para consolidar minha presença digital como **Engenheiro de Computação** e especialista em **Engenharia de Dados e Automação**.

## 🚀 Objetivo
Demonstrar competências técnicas na transformação de processos manuais em fluxos de dados escaláveis e eficientes, destacando resultados reais de otimização operacional.

## 🛠️ Tecnologias e Ferramentas
* **Linguagens:** HTML5, CSS3, JavaScript (ES6+).
* **Design:** Apple-style UI com elementos Cyberpunk.
* **Funcionalidades:** * Internacionalização nativa (PT-BR / EN).
    * Animação de fundo dinâmica via Canvas API (Matrix Effect).
    * Layout responsivo otimizado para dispositivos móveis.
* **Ícones:** Lucide Icons e Simple Icons.

## 📊 Destaques de Carreira (Exibidos no Portfólio)
* **Engenharia de Dados e Automação:** Redução de **87%** no tempo de processamento de tarefas repetitivas (de 40min para 5min) através de scripts em Python.
* **Administração de Bancos de Dados:** Gestão de integridade e execução de queries em ambientes **MySQL** e **SQL Server**.
* **Business Intelligence:** Criação de dashboards em **Power BI** para suporte estratégico à tomada de decisão.
